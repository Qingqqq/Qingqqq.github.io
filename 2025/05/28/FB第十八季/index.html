1-马克思主义-哲学-辩证法

孟子曾有言：“虽有天下易生之物也，一日暴之，十日寒之，未有能生者也。”从哲学角度看，这句话蕴含的哲理是（ ）。

  • A

    事物的发展是量变与质变的统一

  • B

    内因是事物发展的根本原因，是第一位的原因

  • C

    外因对事物的发展起决定性作用

  • D

    外因对事物的发展起着非常重大的作用

本题考查马克思主义。

“虽有天下易生之物也，一日暴之，十日寒之，未有能生者也”意思是即使天下有一种最容易生长的植物，（如果）晒它一天，冻它十天，也没有能存活的。强调了外部环境的重要作用。

A项错误，题干强调的是外部环境对植物生长的重要作用，与量变、质变无关。

B项错误，题干强调的是外因而非内因，选项表述正确，但与题干内容无关。

C项错误，内因是事物变化发展的根本和第一位的原因，对事物的发展起决定性作用。

D项正确，“一日暴之，十日寒之”代表着外部环境（即外因）对事物的重要作用，强调了外因的重要性。

故正确答案为D。

21-经济-宏观经济与调控政策

关于经济常识，下列说法不正确的是（ ）。

  • A

    进一步放宽优化过境免签政策，有助于缩小我国服务贸易逆差

  • B

    LPR下调25个基点，有助于降低居民住房消费负担，促进房地产止跌回稳

  • C

    鼓励和推动消费品以旧换新，有利于扩大内需，推动经济实现良性循环

  • D

    提高存款准备金率，能够有效增加货币供应量，促进经济增长

  • A

    进一步放宽优化过境免签政策，有助于缩小我国服务贸易逆差

    服务贸易逆差是指服务贸易进口额高于出口额。

    要缩小服务贸易逆差，需要增加服务贸易出口，或者减少服务贸易进口。

    进一步放宽优化过境免签政策会促使外国游客来我国旅游、进行商务活动等，这会增加我国旅游服务等相关服务产业的收入，属于增加服务贸易出口，有助于缩小我国服务贸易逆差。

  • B

    LPR下调25个基点，有助于降低居民住房消费负担，促进房地产止跌回稳

    贷款市场报价利率（LPR）是由具有代表性的报价行，根据本行对最优质客户的贷款利率，以公开市场操作利率（主要指中期借贷便利利率）加点形成的方式报价，由中国人民银行授权全国银行间同业拆借中心计算并公布的基础性的贷款参考利率，各金融机构应主要参考LPR进行贷款定价。

    现行的LPR包括1年期和5年期以上两个品种。

    LPR下调后，将推动企业和个人的贷款利率进一步降低，存量和新增房贷利率也进一步降低，将更好地降低居民住房消费负担，促进房地产止跌回稳。

  • C

    鼓励和推动消费品以旧换新，有利于扩大内需，推动经济实现良性循环

    扩大内需就是扩大内部需求，包括扩大投资需求和消费需求。鼓励和推动消费品以旧换新，能降低消费者购买新品的成本，有助于撬动居民消费、扩大内需，推动经济实现良性循环。

错误

  • D

    提高存款准备金率，能够有效增加货币供应量，促进经济增长×

    存款准备金是指金融机构为保证客户提取存款和资金清算需要而准备的在中央银行的存款，中央银行要求的存款准备金占其存款总额的比例就是存款准备金率。

    中央银行可以通过卖出国债、提高法定存款准备金率、提高再贴现率等紧缩性货币政策减少货币供应量，引导利率上升，减少市场上的可贷资金，冷却过热的投资需求，从而抑制通货膨胀。

B正确答案

    危如朝露——温度升高时，朝露会迅速蒸发，从液态转变为气态

    朝露是水蒸气在夜间遇冷凝结成的小水滴，附着在物体表面。在温度升高时会迅速蒸发，从液态转变为气态，该过程属于汽化现象，需要吸收热量。

关于生活常识，下列说法正确的是（ ）。

  • A

    用微波炉加热食物时，不能使用金属容器，是因为金属会吸收微波，导致食物无法变热

  • B

  • C

    被蚊子叮咬后涂抹肥皂水能止痒，是因为肥皂水呈碱性，可中和蚊子分泌的酸性物质

  • D

    天然气报警器应安装在炉灶下方，因为天然气的密度比空气大

C正确答案

  • C

    被蚊子叮咬后涂抹肥皂水能止痒，是因为肥皂水呈碱性，可中和蚊子分泌的酸性物质

    蚊子叮咬人时，会向人体注入蚁酸等酸性物质，这些酸性物质会刺激皮肤，导致皮肤瘙痒。而肥皂水通常呈碱性，在被蚊子叮咬处涂抹肥皂水后，碱性的肥皂水与酸性的蚁酸发生中和反应，从而减轻了酸性物质对皮肤的刺激，达到止痒的效果。

错误

  • A

    用微波炉加热食物时，不能使用金属容器，是因为金属会吸收微波，导致食物无法变热×

    用微波炉加热食物时不能使用金属容器，原因是金属会反射微波，而非吸收微波。微波被反射后，不仅无法有效加热食物，还可能在金属表面产生电火花，甚至引发爆炸等危险情况。

  • B

    水垢的主要成分有碳酸钙和氢氧化镁等，可用食盐水去除×

    水垢主要由碳酸钙和氢氧化镁等组成，食盐水的主要成分是氯化钠，氯化钠与碳酸钙、氢氧化镁不会发生化学反应，无法起到去除水垢的作用。通常会使用酸性物质（如醋酸等）去除水垢。

  • D

    天然气报警器应安装在炉灶下方，因为天然气的密度比空气大×

    天然气的主要成分是甲烷，甲烷的相对分子质量约为16，而空气的平均相对分子质量约为29。在相同条件下，气体的密度与气体分子的相对分子质量成正比，所以天然气的密度比空气小。一旦发生天然气泄漏，气体会向上扩散，因此天然气报警器应安装在炉灶上方，以便能及时检测到泄漏的天然气。

    交叉配血是在输血前必须进行的一项重要试验，用于检测供血者和受血者的血液是否相兼容，以确保输血安全。虽然通过ABO血型和Rh血型等常规血型鉴定可以初步判断血型，但可能存在血型亚型或其他不规则抗体等情况，交叉配血可以进一步确认供血者和受血者的血型是否完全匹配。

下列有关燃油污染的说法不正确的是（ ）。

  • D

    燃油燃烧产生的尾气会加剧酸雨的形成

B正确答案

  • A

    燃油泄漏到土壤中，会降低土壤的透气性

    燃油是由多种有机化合物组成的混合物。当燃油泄漏到土壤中，会迅速在土壤颗粒表面铺展。由于其疏水性，会排斥水分子并形成连续或半连续的油膜，紧密附着在土壤颗粒表面，阻碍土壤与空气的气体交换，从而降低土壤的透气性。

  • C

    燃油中含有苯，泄漏后会挥发到空气中

    燃油中通常含有苯等有机化合物。苯具有挥发性，燃油泄漏后，苯会挥发到空气中造成空气污染，长期吸入苯等挥发性有机化合物可能会导致血液系统疾病，甚至会增加患癌症的风险。

  • D

    燃油燃烧产生的尾气会加剧酸雨的形成

    燃油燃烧会产生二氧化硫、氮氧化物等污染物，这些污染物排放到大气中，会与空气中的水汽等发生化学反应，形成硫酸、硝酸等酸性物质，随雨水降下形成酸雨。

错误

  • B

    燃油泄漏到水体中，会造成水体富营养化×

    水体富营养化是指水体中氮、磷等营养物质含量过多，导致藻类等浮游生物大量繁殖、水质恶化等现象。

    燃油的主要成分是碳氢化合物，并不含有大量的氮、磷等营养物质，不会直接造成水体富营养化。燃油泄漏到水体中，会在水面形成油膜，阻碍氧气进入水体，造成水中生物缺氧死亡。

第一步：找出论点和论据。

论点：辣椒碱有望成为治疗急性酒精性胃炎的有效药物成分。

论据：辣椒中的辣椒碱能够激活胃内的“胞质接头蛋白Keap1-转录因子Nrf2-抗氧化反应元件Are”通路，这是人体最重要的抗氧化系统之一，从而预防酒精所致的急性胃黏膜损伤（即酒精性胃炎）。进一步的研究还表明，由白蛋白包被的辣椒碱纳米制剂在酒精导致的急性胃黏膜损伤模型中表现出卓越的抗氧化和抗炎效果。
